# projeto-guessnumber

O GuessNumber é um browser Gaming feito com as tecnologias HTML, CSS e JavaScript. O objetivo do game é acertar o número no qual o navegador está "pensando", esse número varia de 0 a 10 e suas tentativas são contadadas, assim que você acertar o número pode jogar novamente clicando no botão: "Jogar Denovo"
